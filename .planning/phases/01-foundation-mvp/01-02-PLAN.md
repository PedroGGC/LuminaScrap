## Threat Model

- **IP blocking:** High-frequency requests could trigger anti-bot — respect rate limits, use delays
- **Data quality:** Invalid data from scrapers — validate with Zod before DB insert

## Dependencies

- `axios` or `fetch` for HTTP
- `zod` for validation
- `drizzle-orm` for DB writes

## Success Criteria

- [ ] Kabum scraper extracts product data successfully
- [ ] ML scraper extracts product data successfully
- [ ] Scraper errors logged but don't stop other sources
- [ ] GHA workflow runs on schedule
- [ ] Data written to Neon with valid schema
