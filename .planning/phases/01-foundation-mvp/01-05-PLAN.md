# Plan 01-05: Next.js Frontend + TanStack Table + Vercel Deploy

**Phase:** 1 (Foundation & MVP)
**Status:** Pending execution

## Objective

Build the Next.js frontend with TanStack Table, shadcn/ui components, filter bar, and deploy to Vercel.

## Requirements Covered

- UI-01: Interactive table with columns (name, brand, CPU, GPU, RAM, storage, price, store, updated)
- UI-02: Combined filters (CPU, GPU, RAM, brand, price)
- UI-03: Text search
- UI-04: Column sorting
- UI-05: URL param state for filters
- UI-06: Loading skeleton
- UI-07: Badges for GPU, brand, category

## Threat Model

- **Data exposure:** Frontend public — ensure API handles queries efficiently
- **Performance:** Large datasets — implement pagination, avoid over-fetching
